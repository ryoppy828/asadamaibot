# 卓球AIアシスタント Q&Aウェブサイト (Walkthrough)

## 概要
YouTubeチャンネル（@tabletennis_mai）の動画情報に基づいて、ユーザーの質問に答えるAI Q&Aサイトの構築・機能実装が完了しました。本ドキュメントでは、実装した機能やデザインの変更点についてまとめます。

## 実装した機能・変更点

### 1. システム要件とアーキテクチャ
- **フロントエンド・フレームワーク**: Next.js (App Router) を採用し、スタイリングには Tailwind CSS を使用しました。
- **バックエンド・API連携**: Gemini API と連携するバックエンド処理を実装し、AIによる高度な回答の生成を可能にしました。
- **診断機能の組み込み**: サービス開始の初期機能として、「技術相談」「MYTEMグッズ診断」「用具診断」の3つのモードを実装しています。

### 2. データ取得とナレッジベース
- **YouTubeデータ連携**: @tabletennis_mai の全動画の情報を取得・解析し、AIが回答を生成する際のベース知識（コンテキスト）として組み込みました。

### 3. チャットAIの構築（人格・振る舞い設定）
- **推論モデル**: Google Gemini 2.5 Flash モデルを採用し、高速かつ精度の高い応答を実現しています。
- **AIアシスタントの人格**: システムプロンプトを用いて、卓球チャンネル「@tabletennis_mai」の公式AIアシスタントである**「ぴんぽんわんこ」**として振る舞うよう設定しました。明るく元気で、かわいいわんちゃんのようなトーンでユーザーと対話します。
- **関連URLの提示ロジック**: アシスタントは回答の最後に、質問の種類に応じた関連URLを必ず提示するようプロンプトを制御しています。
  - **技術相談**: 関連するYouTubeチャンネル（@tabletennis_mai）の該当動画URL
  - **MYTEMグッズ診断**: [MYTEM公式ショップ (https://mytem.official.ec/)](https://mytem.official.ec/) のおすすめ商品URL
  - **用具診断**: 該当するAmazonの商品リンク

### 4. UI/UXデザインの刷新
イマドキでおしゃれな印象を与えるため、以下のデザイン要件に基づきUIを再設計しました。
- **カラースキーム**:
  - ベース背景: 薄桜色のグラデーション（`#fff5F8`系）
  - メインカラー: 洗練されたピンク（`#ff8fb3`）
  - テキストカラー: チャコールブラウン（視認性と柔らかさを両立）
- **Glassmorphismの導入**: 半透明のぼかし効果（グラスモーフィズム）を随所に取り入れ、立体的でモダンな質感を演出しました。
- **対話インターフェース**: チャットの入力ボックスを画面中央下部に配置し、大きく丸みを持たせたピル型のデザインを採用することで、ユーザーが直感的に操作しやすいレイアウトを実現しています。
